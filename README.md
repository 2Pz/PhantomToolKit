# PhantomToolKit

An experimental web-based utility mod for Elden Ring to manage saves, swap builds, inspect players, and toggle cheats. Supports Windows and Linux.

**Note:** This project is highly experimental using `fspy`, a custom Python library I made using [`PyO3`](https://github.com/pyo3/pyo3) to bind specific functions from [`fromsoftware-rs`](https://github.com/vswarte/fromsoftware-rs).
